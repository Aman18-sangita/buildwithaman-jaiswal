# Aman Jaiswal — Portfolio

A modern, fully responsive personal portfolio built as a single static HTML file — no frameworks, no build step, instant deploy.

**Live:** [https://buildwithaman-jaiswal.vercel.app/]

## Overview

Personal portfolio showcasing my work as a Software Engineer and Cybersecurity Researcher — covering technical skills, full-stack & AI projects, published research, and competitive programming stats.

## Highlights

- Light/dark mode with persistent theme
- Animated hero, scroll-reveal sections, and interactive carousel
- Embedded resume — one-click download, no external hosting
- Fully responsive across desktop, tablet, and mobile
- Zero dependencies — pure HTML, CSS, and vanilla JS

## Tech Stack

HTML5 · CSS3 (Grid/Flexbox, custom properties) · Vanilla JavaScript · Google Fonts

## Deployment

Hosted on **Vercel** as a static site. Pushes to `main` auto-deploy.

## Run Locally

```bash
git clone https://github.com/Aman18-sangita/aman-portfolio.git
cd aman-portfolio
open index.html
```
